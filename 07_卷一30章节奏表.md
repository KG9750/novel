# 《世界模型：一秒先机》卷一30章节奏表

## 使用说明

本表是写正文时的执行表。`01_卷一大纲.md`负责卷一整体结构，本文件负责逐章留存：每章必须有即时目标、具体阻碍、局部结果和章尾钩子。

单章建议字数：2000到2500字。

基础章法：

```text
承接上章问题 -> 当章目标 -> 阻碍升级 -> 主角决策 -> 局部结果 -> 代价/反转 -> 章尾卡点
```

## 写作检查

每章写完后检查：

- 主角这一章有没有明确目标？
- 目标有没有遇到具体阻碍？
- 能力有没有带来爽点或代价？
- 这一章有没有局部结果？
- 章尾是否让读者想点下一章？
- 是否连续两章都在解释设定？如果是，必须加行动或冲突。
