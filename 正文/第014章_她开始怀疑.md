# 第014章 她开始怀疑

北辰三院值班室，晚上八点。

林夏把北辰路站事故监控暂停在第十七秒。

画面里，人群还没有彻底倒向车门，严存已经侧身去拉那个小女孩。

她反复看了五遍。

第五遍时，她没有再看人群，只看严存。

十六秒，他还被挤在车门边。

十七秒，他抬头，肩膀先转。

十八秒，人群才撞上来。

正常反应不会这样。正常人会先听见尖叫、先看见人倒，再伸手。严存像是先知道那里会塌。

电脑屏幕的冷光照在林夏脸上，旁边放着三号床老人的抢救记录。她刚才已经把老人用药、血压波动、过敏史和监护仪报警前的护理记录看了三遍。

老人那次抢救，她判断得快，是因为药物、年龄、呼吸和唇色都在往同一个方向推。

严存不一样。

他在地铁里，没有病历，没有监护仪，也没有完整视野。

林夏关掉监控。

手机屏幕亮起。

严存发来消息：“检查结果出来了，动态心电正常。别再让我做项目外体检。”

林夏盯着“正常”两个字看了几秒，回他：“晚上来医院拿报告。”

严存回得很快：“发电子版。”

林夏：“纸质版需要本人签字。”

这句话是假的。

她就是想当面看他一次。

晚上八点半，严存到了北辰三院急诊外走廊。

他穿着公司那件皱了的夹克，眼下发青，手里的手机一直震，显然还在处理客户小单。林夏站在护士站旁，把一只牛皮纸袋递给他。

严存伸手来接。

林夏故意松手。

纸袋落下的瞬间，严存的手已经往下一抄，稳稳接住。

他抬眼看她。

林夏说：“你提前了。”

严存把纸袋拿稳：“你故意的。”

“对。”林夏没有绕，“刚才如果是正常反应，你应该先看纸袋掉，再伸手。可你是先伸手，纸袋才掉。”

严存沉默了一下：“你想证明什么？”

“证明低血糖、应激、巧合都兜不住你。”

急诊走廊里有人推着轮椅经过，橡胶轮碾过地面，发出低低的擦声。

严存把纸袋夹在臂弯：“林夏，有些事你别掺和。”

“我已经掺和了。”她看着他，“你在我的医院做检查，手上还缠着我包的纱布。”

严存转开视线。

林夏更火：“你每次都这样。出事先扛，扛不住再装没事。”

这句话比质问更狠。

严存刚要开口，抢救室门忽然被撞开。

一个年轻男人被推床送进来，胸口衣服被剪开一半，脸色灰白。家属跟在后面哭，护士一边推一边喊：“车祸伤，意识模糊，血压九十六！”

急诊的节奏瞬间变了。

林夏转身过去。

严存本能地看向推床。

他没有看见明显残影。

男人的手垂在床边，随着推床晃动。输液袋还没挂上，监护仪探头刚夹住手指，数字正在刷新。

林夏的目光落在男人胸口。

擦伤、淤青、胸廓起伏不对称，吸气时右侧明显慢了半拍。

“别直接推进去。”林夏一把按住推床边缘，“先做胸腔评估。”

护士急道：“林医生，血压还能维持，先送创伤室！”

“他右侧呼吸音弱。”林夏声音发紧，“备穿刺包，通知创伤室，路上别耽误。”

几乎就在她说完的下一秒，年轻男人猛地张口，发出一声吸不上来的气音。

监护仪的血氧数值直线往下掉。

护士脸色大变：“张力性气胸！”

走廊里乱成一片。

严存看着林夏抓起穿刺包，手在抖，但动作没有慢。

她不是凭空报答案。擦伤的位置、胸廓的起伏、病人那口吸不上来的气，全都摆在现实里。林夏只是比别人更敢相信自己的判断。

抢救室门关上的瞬间，她回头看了严存一眼。

那一眼没有神秘，也没有惊喜。

只有疲惫和怒气。

像在说：你明明也知道危险，却还想一个人扛。

严存站在走廊边，手机忽然震了一下。

急诊科护士发来一段语音，声音压得很低。

“林医生，刚才有个穿黑夹克的人问三号床老人和车祸病人的抢救记录，还问严存是不是在我们这里做过检查。”

林夏推开抢救室门，脸色一点点冷下来。

她开始怀疑的，不只是严存。
