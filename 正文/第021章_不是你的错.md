# 第021章 不是你的错

北辰三院急诊外，下午五点二十。

“不是你的错。”

林夏把这句话压得很低，低到只有严存听见。

长椅冰凉，严存手里还攥着复盘会的资料袋。袋角被他捏出一道白痕，里面压着北辰路站原始预警、阈值改动记录、钱永胜审批意见和客户方重查函。

纸页越厚，他心里越沉。

它们能把锅从他身上挪开，却挪不开那几条曾经亮起的预警。

如果他早一点顶上去呢？

如果那天他没有把工作、工资、父亲的住院费全压在沉默后面呢？

急诊门一开，消毒水味和人声一齐涌出来。担架车从走廊尽头推过，家属跟在后面，鞋底拖过地面，乱得像一把被揉碎的纸。严存抬头的瞬间，眼前又多了一层残影。

事故已经过去不止三天。最初那种满世界都拖着重影的余震退了，可一旦危险逼近，残影仍会像被重新点燃的线，猛地烧到眼前。

担架车右前轮会卡住。

下一秒，车身会偏，吊瓶架会砸向旁边的小孩。

严存身体比脑子先动。他伸手扣住吊瓶架，另一只手顶住担架车侧沿。现实里，右前轮果然在地砖缝里一顿，护士没推稳，车头斜了半寸。

“小心！”

吊瓶架被严存按住，没有砸下去。

护士惊了一下，连忙道谢。家属只顾着哭，没人多看严存一眼。可林夏看见了。

她站在抢救室门口，白大褂袖口卷到小臂，眼下有很淡的青色。她没有问他为什么又提前动了，只走过来，按住他的手腕，把他从担架车旁边拽开。

“你刚才又用了？”

严存把手抽回来：“顺手。”

“严存。”林夏盯着他，“顺手不会让你手心全是汗。”

严存低头，才发现指尖在发抖。

复盘会结束后，他本该回公司签协查记录。可客户代表一句“北辰路站系统需要医院侧伤情数据”，把他带到了北辰三院。顾南枝的人没有出现，钱永胜也暂时没声。所有人都说他赢了一局。

他只觉得脑子里那根线绷得更紧。

他能看见一秒。

可他看不见已经发生的那一秒。

“资料我看过。”林夏说，“你当时只是外包测试员，权限被卡，报告被压，预警不是你关的。”

严存笑了一下：“你以前不是最烦我给自己找理由？”

林夏没有退：“我烦的是你明知道该争还装没看见。现在你已经争了。”

这句话像针，扎得不重，却准。

他们以前吵得最凶的一次，也是在医院。

那天父亲第一次追加治疗方案，严存把账单折了又折，说公司项目快结款了。林夏问他为什么不去找主管要被扣的绩效，他说算了，年底考核还在人家手里。林夏当时看了他很久，只说了一句：你有能力，你只是怕撕破脸。

严存那时没接话。

他以为沉默能把争执拖过去。

后来他们分手，也像一场没有爆发的事故。没有谁出轨，没有谁背叛，只是每一次该开口的时候，他都慢半拍。

现在他多出来一秒，才发现提前看见危险只算开头。

伸手之后，后果才真正落下来。

严存看向抢救室门口。红灯亮着，里面有人在喊除颤，有人报血压，有人推药。北辰三院的急诊从来不会停，像一条吞人的流水线，谁慢半拍，谁就被卷走。

“我在会上赢了。”严存说，“可北辰路站那天，他们没赢。”

林夏沉默了一秒。

她听懂了。

严存继续说：“我现在看什么都像风险。车轮、门缝、手、玻璃杯、台阶。只要看见，我就想拦。可我拦不完。”

“你当然拦不完。”林夏说，“医生也救不完所有人。”

严存看她。

林夏的眼神很稳：“但这不等于每一个没救回来的，都是医生杀的。”

走廊里忽然传来一声尖叫。

一个中年男人扶着墙，脸色发白，嘴里说着“没事”。他旁边的女人还在挂号机前翻病历，根本没意识到危险。严存看过去，只看见男人身体重心会往左倒，后脑撞向金属座椅扶手。

他刚要上前，林夏已经动了。

她比严存更快。

她没有直冲男人身后，而是先抓起护士站旁的氧气袋，抬手指向轮椅：“推过来！快！”

男人还没倒，林夏已经一手托住他的下颌，一手摸向颈侧。

下一秒，男人膝盖一软。

轮椅及时卡到他身后。

护士冲过来，监护夹扣上手指。屏幕上的血氧数字从九十六往下掉，九十三、八十九、八十四。

林夏的声音压得很快：“急性过敏可能，准备肾上腺素，开静脉通道。”

女人这才反应过来，脸瞬间白了：“他刚才还说没事！”

严存站在原地。

他看见的只是男人会摔。

林夏盯住的却是他的唇色、喉间吸气和颈侧塌下去的那点脉动。

她抓的是即将断掉的呼吸。

抢救推车撞开人群，轮子压过地面，发出一串急响。林夏跟着护士把男人推向处置室，经过严存身边时，她脚步忽然顿了一下。

她的眼睛里闪过一瞬茫然。

那不是抢救后的疲惫。

她也被自己刚才那一下吓到了。

她按着处置室门框，指节发白，像太阳穴里被细针扎了一下。

那一瞬，严存从她脸上看见了自己第一次走出地铁站时的表情。

没有惊喜。

只有被迫知道后的发冷。

林夏低头看了一眼自己的手。指尖还有消毒液的味道，可掌心很凉。刚才她甚至没有经过完整判断，男人的唇色、喉间的吸气声、颈侧血管的塌陷，在她眼前像被提前拉成了一条线。

那条线往下坠，她就伸手去抓。

抓住了，病人活。

抓不住，她可能会比家属更早看见那条线沉下去。

这种事，没人教过她怎么接受。

严存抬手想拉她，却只碰到她白大褂的衣角。

处置室门合上。

门缝里，监护仪尖锐地响了一声。

严存看着那道门，忽然明白顾南枝为什么一直不急着解释。

因为异常不只在他身上。

几分钟后，林夏从处置室里出来，手套上沾着一点血，声音比刚才更低。

“严存。”

她看着自己的手。

“刚才，他还没喘不上气，我就看见了。”
