# 第035章 小型演练

北辰路站地面控制室，上午八点二十。

演练还没开始，站厅里已经站了三十多个工作人员。

黄马甲、对讲机、警戒带、临时引导牌，全都摆好了。运营方负责人拿着流程表，反复强调只是小型演练，不开放乘客，不做媒体展示。

严存站在玻璃窗后，看着下方空荡的站厅。

空站比早高峰干净太多。

也更容易骗人。

顾南枝在旁边说：“今天不是考你救人。考你能不能把看到的东西翻译成流程。”

严存没接话。

他昨晚只睡了四个小时。林夏早上发来消息，只有四个字：记停手线。

演练开始。

第一轮模拟的是 A 口封闭后，乘客从 B 口和 C 口改道。站务人员按流程摆牌，广播按稿子播，屏幕上人流模拟数据平稳得像一张假笑脸。

严存看着站厅中段。

一名扮演乘客的工作人员推着婴儿车，从导流带旁边绕了一步。另一个工作人员低头看手机，从立柱后出来。

残影闪了一下。

下一秒，婴儿车前轮会卡住警戒带底座，后面那个人会本能避让，肩膀撞向临时指示牌。牌子倒下，不伤人，但会挡住 C 口通道。

这不是大事故。

可它会让整条动线在演练里乱起来。

严存按下对讲：“C 口导流带底座外移二十厘米，指示牌从立柱左侧挪到右侧。手机低头人员从第二轮开始列入干扰项。”

运营负责人皱眉：“现在动线没问题。”

话音刚落，婴儿车前轮一顿。

那名工作人员脚步乱了一下。

指示牌被肩膀碰得晃动，差一点倒下。

控制室里安静了半秒。

顾南枝没有看严存，只对负责人说：“按他说的改。”

第二轮开始，动线顺了很多。

严存没有再急着报每一个小问题。

他把视线从人脚下移到出口边缘，从每个动作里挑出会影响整体流向的一两个点。其余那些不伤人、不堵路的小碰撞，他强迫自己放过去。

第三轮，一个临时保洁车从服务通道推出来。

保洁员没按演练脚本走。

残影里，保洁车左轮会压住警戒带，整条带子弹起，前排三个人会同时停步。停步会传到后面，形成一个小反堵。

严存立刻开口：“服务通道暂停出车。”

顾南枝看了他一眼。

运营负责人这次没再问为什么，直接对讲：“保洁车停。”

车停在警戒带前半米。

反堵没有发生。

演练结束后，负责人把流程表翻到最后一页，在“外部协查意见”一栏写了三条。

底座外移。

低头干扰项加入脚本。

服务通道与导流时段错开。

他写完，主动朝严存伸手：“今天这三条有用。”

严存握住那只手，掌心还潮着。

顾南枝的手机同时亮起。

一条内部评估推送出来。

临时协查对象：严存。

评估状态：可参与低等级公共安全演练。

严存刚松一口气，手机又震。

陌生号码发来一张照片。

照片里，刚才那辆保洁车停在通道口，车把上挂着一枚黑色钥匙扣。

下面一行字：

“你只看见它会撞上去。”
