# 《一秒先机》项目定位

## 核心定位
- 题材：都市职业爽文 + 微异能 + 轻悬疑 + 单女主慢热
- 核心卖点：普通社畜只比别人快一秒，却能靠这一秒完成别人翻不了的盘
- 平台方向：番茄男频长线连载，目标 120 万字以上
- 爽点结构：受压制 -> 小翻盘 -> 被抢功 -> 再翻盘 -> 副业起势 -> 创业掌局

## 对外包装
- 书名 A：一秒先机
- 书名 B：我能看见一秒后的轨迹，从背锅运营到资本新贵
- 一句话卖点：所有人都只能看见结果，而他能看见每个动作一秒后的落点。
- 标签建议：都市脑洞、职场逆袭、单女主、创业、商业博弈、轻悬疑

## 核心设定
- 男主：许澈，28 岁，互联网增长运营，能力觉醒前是标准背锅社畜
- 女主：林知夏，27 岁，交互设计师，专业硬、嘴硬、执行力强
- 能力：轨迹预读 1 秒
  - 只对正在运动的目标生效
  - 默认可预读 1 秒，极限 1.3 秒
  - 同时追踪目标越多，副作用越强
  - 副作用包括头痛、耳鸣、残影、恶心、注意力过载
- 世界观边界：不引入超自然组织，不全民觉醒，不公开异能，保持现实都市底色

## 读者抓手
- 第一章强画面：地铁急刹，整座站台拖出淡色轨迹线，男主提前半步救下女主
- 第三章第一次兑现：用户测试里看见鼠标未来轨迹，赌一把改版成功救活项目
- 前 30 章三大爽点：
  - 救下女主
  - 用能力把濒死项目救活
  - 让抢功上司在公开场合第一次吃瘪

## 长线路径
1. 公司翻盘：从背锅人变成项目核心
2. 副业咨询：把能力变成现金
3. 方法论成型：从“我会”变成“团队也能学”
4. 离职创业：与女主共同搭建产品与服务
5. SaaS 化：将经验封装为工具系统
6. 资本局：融资、并购、行业对打，彻底完成身份跨越

## 写作原则
- 先给冲突，不先讲设定
- 每章都要有当前目标和章末问题
- 前 50 章事业线 7，感情线 3
- 女主必须有专业作用，不做纯陪衬
- 能力升级只提升使用精度，不改成万能挂
