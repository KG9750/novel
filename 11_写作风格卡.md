# 《世界模型：一秒先机》写作风格卡

## 使用目标

借鉴《笑傲江湖》的三项长处：叙事流畅、画面清晰、分镜分明。

只借技法，不搬古腔。场景仍然是当代都市、地铁事故、职场追责、医院急救、系统调查和小队任务。

每一章都要先让读者看见事情正在发生，再理解事情为什么发生。

番茄样本期还要额外满足：手机端读起来快、段落短、冲突早、章尾硬。文笔服务留存，不用长段漂亮描写换掉行动推进。

## 参考抽样

参考《笑傲江湖》的几个代表场面：

- 第一章《灭门》：福威镖局危机四起，小酒店、门板、匕首、旗杆、尸体消息接连压来，局势一层比一层紧。
- 第六章《洗手》：刘正风金盆洗手本是公开仪式，场面先稳，后堂家眷忽被押出，厅上形势瞬间翻面。
- 第十章《传剑》：令狐冲与田伯光对拆，再到风清扬点破“无招”，动作、讲解、顿悟三线并行。

这些段落最值得借的不是武侠词汇，而是场面调度、动作推进和视角切换。

## 核心技法

### 0. 转场先给地点和时间

主要场景切换时，先用一句简明信息锚定地点和时间，再进入动作。

可用格式：

- 北京，五道口，周二早高峰。
- 北辰三院急诊，晚上十点四十。
- 穹图智行十七层会议室，上午九点十二分。

要求：

- 地点和时间只负责锚定，不展开解释。
- 城市、区域、机构、楼层、时间段按场景重要性取两到三个即可。
- 不要每个小动作都标时间；只在主要场景切换、章首、重场面开始时使用。

### 1. 先摆局，再落人

- 先让读者知道场面上有哪些人、站位如何、危险从哪里来。
- 再把镜头压到某一个动作、某一句话、某一个器物上。
- 这样读者不会迷路，冲突也会更有方向感。

适配到本书：

- 车厢先写车门、人流、站台缝隙，再落到谁脚下一滑。
- 会议室先写投影、座位、谁掌话语权，再落到谁把责任报告推过来。
- 医院先写抢救床、监护仪、走廊人声，再落到某个即将崩掉的生命体征。
- 任务现场先写入口、出口、盲区、摄像头，再落到严存看见的那一条危险轨迹。

### 2. 动词带动镜头

- 少写“非常紧张、压迫感很强、气氛凝重”。
- 多写“车门一震、灯管一闪、报告拍到胸口前半寸、监护仪尖响、鼠标先偏了半寸”。
- 一连串动作要有先后，不要平铺。
- 日常场景不要高频制造致伤巧合。玻璃碎片飞向颈侧、尖锐物刺向要害这类偶发危险不能一天内反复出现；危险应来自场景既有压力、人群惯性、流程漏洞或反派行动。

可用动词方向：

- 推、压、撞、扯、扣、偏、掠、砸、收、顿、转、避、封、卡、逼。

### 3. 冲突靠动作、对话和反应循环推进

推荐节奏：

1. 一个人出手或开口。
2. 另一个人接招或失手。
3. 周围人安静、惊呼、躲闪、交换眼神。
4. 严存心里只闪过一句判断。
5. 下一轮冲突立刻接上。

不要让大段心理独白拖住动作场。心理描写只负责校正局势，不代替事件。

### 4. 远景、中景、特写之间快速切换

现代场景可直接套用：

- 远景：整节车厢、整间会议室、急诊走廊、任务指挥室。
- 中景：严存、林夏、钱永胜、顾队、小队成员。
- 特写：裂开的手机屏、责任报告、监护仪数字、快要按下去的鼠标、车门灯。
- 反打：其他人的目光、群消息、监控慢放、短暂沉默。

一场戏的理想镜头顺序：

```text
全场局势 -> 关键人物 -> 关键动作/物件 -> 群体反应 -> 回到主角判断
```

### 5. 关键场面要重墨

每章最多一个真正的重场面。

平常段落可以简写；到了翻盘、救险、揭锅、公开对峙、任务转折，就要把动作细节和对话密度写上去。

重场面结束后，立刻给后果，不拖。

### 6. 心理描写只点一下

心理描写优先写判断，不优先写抒情。

推荐：

```text
他立刻明白，这锅已经扣死了。
```

不推荐：

```text
他感到一种深沉而复杂的委屈。
```

## 语言层面的借法

### 要借的

- 白话为主，句子顺，不故作玄虚。
- 用少量有力度的书面词压住场面，比如“骤然、陡然、逼近、横插、压下、掀开、收住”。
- 对话简洁，能一句顶一句。
- 叙述常带空间方向和身体方向，让读者知道谁朝哪边动。

### 不要借的

- 不用“在下、贫道、阁下、此人、须知、岂料”这类古典口吻。
- 不刻意模仿旧武侠的比喻和成语密度。
- 不让所有人物都说得像江湖人。
- 不把办公室写成古代门派，只借其权力结构和对峙手感。

## 本书专项迁移

### 1. 把能力写成临场判断，不写成设定播报

- 先写异样的视觉、听觉或身体反应。
- 再写严存捕捉到的下一秒结果。
- 不先解释规则，先让规则在危险里兑现。

第一章能力触发要避免“屏幕里的线跑进眼睛”这种玄幻感。更适合写成：

```text
他做了三年人流预测，太熟悉这种场面。谁的重心偏了，哪股人流要挤，哪只脚会踩空，他有时候看一眼就能猜个七八分。

可猜，是猜。

这一次不一样。
```

### 2. 把职场写成拆招

- 上司逼签报告，不只是语言攻击，而是一次先手。
- 严存接招、缓冲、留证、反打，都是拆招。
- 日志、预警记录、监控慢放、权限和责任报告，都要写出手感。

### 3. 把感情线写成并肩临场

林夏和严存的关系推进，优先放在一起扛事、一起判断、一起救人的场景里。少靠纯抒情对话。

### 4. 把小队写成多层现实拼图

第一卷不让小队实际登场。第二卷后，小队不是“严存带挂件”。每个人都只能看自己领域的一秒。

任务场面要写出信息交叉：

- 严存看见运动轨迹。
- 链路成员看见系统断点。
- 风控师看见资金风险。
- 谈判专家看见心理反应。

真正的爽点是这些信息在同一秒对齐。

### 5. 把章末写成势未收

这一章的动作可以结束，但读者必须知道下一章会有更大的对手、更硬的代价，或更险的局面。

顾队暗线、小队伏笔、能力副作用，都适合放在章末轻轻一压。

## 可直接执行的章节模板

### 开场 150 到 300 字

- 直接给异常。
- 直接给空间。
- 直接给当下目标。

例子：

- 乘客还没倒，严存先看见后脑撞地。
- 责任报告还没拍到胸口，他先看见钱永胜下一秒会把录音笔推近。
- 病人还在说没事，林夏先从唇色、呼吸和用药记录判断风险。

### 第一段冲突

- 主角还没完全明白，但已经被卷入。
- 先让他被动接招。
- 用 2 到 3 个连续动作把局势拉紧。

### 第二段冲突

- 主角开始看见别人看不见的下一步。
- 这一段要写得像拆招，不要写成能力播报。

### 第三段冲突

- 对手或环境加码。
- 群体反应进入。
- 镜头切回特写，给读者一个最具体的危险点。

### 结尾

- 当前场面先落。
- 立刻抛出下一轮压力。

## 番茄移动端风格要求

- 前 300 字内给场景、主角、危险或利益冲突。
- 单段尽量控制在 1 到 4 行手机阅读长度，重场面可短句加速。
- 对话一句顶一句，少用解释型长对话。
- 每章只讲一个核心目标，避免一章里同时开三条新线。
- 设定解释不超过当前场面所需，解释后立刻验证或反例。
- 章尾最后 200 字必须压出新问题，不用空泛感叹收尾。

## 平台避坑语感

| 场景 | 推荐写法 | 避免写法 |
|---|---|---|
| 地铁事故 | 北环支线、北辰路站、虚构系统故障 | 对应真实线路、真实事故细节 |
| 医疗现场 | 林夏基于经验做剧情判断 | 现实急救教程式步骤 |
| 金融/灰产 | 风险链、账户异常、反派诱惑 | 可复制的违法操作 |
| 国家机构 | 架空时序处、联合异常应对 | 影射真实部门内部流程 |
| 能力解释 | 他看见下一秒轨迹，代价是耳鸣鼻血 | 大段科学论文式原理 |

## 运营型章尾写法

章尾不是硬断句，而是把读者推到下一个判断。

可轮换类型：

- 外部确认：他不是反应快，他是提前动了。
- 匿名警告：别再用你的眼睛看下一秒。
- 关系逼问：你到底看见了什么？
- 利益诱惑：钱到账的同时，更高价邀约来了。
- 规则反例：他看见了轨迹，却救不了那个人。
- 卷尾升级：三份封存档案打开他看不见的三层现实。

每章写完后补一个“读者会评论什么”。如果想不出来，说明本章冲突还不够具体。

## 章回自检清单

- 开头三段内，是否已经出现明确冲突？
- 读者是否随时知道人物站位、视线和动作方向？
- 是否至少有一个可以拍成镜头的细节特写？
- 是否用了动作、对话和反应来推进，而不是只靠解释？
- 主角心理是否克制，没有淹没动作？
- 本章是否只有一个真正的重场面？
- 结尾是否留下明确的下一步问题？

## 调用口令

以后写正文时，可以直接使用这句：

```text
按《11_写作风格卡》写这一章。借《笑傲江湖》的流畅推进、画面感和分镜感，但保持现代都市语言，不写成古装腔。
```

更具体一点：

```text
本章重点借：先摆局再落人 / 动作带镜头 / 对话与反应循环推进 / 结尾势未收。
```
