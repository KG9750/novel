# 《世界模型：一秒先机》卷一50章节奏表

## 使用说明

本表负责逐章留存：每章必须有即时目标、具体阻碍、局部结果和章尾钩子。

基础章法：

```text
承接上章问题 -> 当章目标 -> 阻碍升级 -> 主角决策 -> 局部结果 -> 代价/反转 -> 章尾卡点
```

第一卷硬规则：前 50 章只出现严存一个特殊能力者；林夏不具备能力；其他能力者只在卷尾以封存档案引出。
